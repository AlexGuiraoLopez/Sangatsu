--Sangatsu--

Descripción del programa:
=> Simulación del clásico juego "Tres en raya" para la consola del sistema operativo.

=> Selecciona el modo de juego (Un jugador o multijugador). En el caso del modo un jugador
selecciona la dificultad de la CPU.

=> Alinéa tres casillas antes que el rival para ganar la partida.


Instrucciones de ejecución:
=> Desde la consola del sistema sitúate en el directorio "dist" con el comando cd + "ruta del directorio"
 y ejecuta el comando "java -jar Sangatsu.jar".

Tecnologías utilizadas:
=> Java

Versión 1.0.
